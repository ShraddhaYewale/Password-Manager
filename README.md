## About the Project

Creating strong passwords is often overlooked, yet it is essential for online security. This application allows users to generate random passwords based on selected options such as character types and length. The goal of this project is to provide a practical tool while reinforcing JavaScript fundamentals and frontend concepts.

---

## Features

- Generate random and secure passwords
- Option to include uppercase letters, lowercase letters, numbers, and symbols
- One-click copy to clipboard
- Simple, clean, and user-friendly interface

## How It Works

1. Open the application in your browser.
2. Select the desired password options.
3. Click the **Generate Password** button.
4. Copy the generated password using the **Copy** button.
